Welcome To AddressBook!

A simple addressbook using nodejs, where you can use the following endpoints to check API's output.

To create conatct by post use this syntax: /api/create-contact Give it a try.
you can also use GUI form to create contact via:/api/create
To fetch all contact list with phase{name|city|state} match and limit result using pagination:/api/fetch/:phaseName/:limitCount/:pageNo Give it a try.
To delete single contact use:/api/delete-single/:id.Give it a try
To fetch single contact from adressbook:/api/fetch-single/:id. Give it a try
To fetch all contact list from adressbook:/api/fetchall. Give it a try
